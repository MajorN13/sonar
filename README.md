# Sonar — Моделирование и фильтрация эхолокационной сети датчиков

Проект реализует физическую симуляцию сети гидрофонов на верёвках, прикреплённых к кораблю, с последующей обработкой данных эхолокации: триангуляцией положений датчиков, фильтрацией Калмана и анализом погрешностей.

## Структура проекта

```
sonar-main/
├── program/
│   ├── Simulation.py            # Физическая модель и анимация сети датчиков
│   ├── Kalman.py                # Расширенный фильтр Калмана (EKF) для сети
│   └── Error.py                 # Анализ погрешностей, DOP, Монте-Карло
├── Animations/
│   ├── sonar_basic.mp4          # Симуляция движения сети датчиков
│   └── sonar_triangulation.mp4  # Симуляция с предсказанием положений
├── images/                      # Результаты экспериментов и графики
├── presentation.pdf             # Пояснительная записка / отчёт
└── presentation.tex             # Исходник презентации в LaTeX
```

## Что делает каждый модуль

### `Simulation.py`
Центральный модуль. Содержит физическую модель верёвок, эхолокационную модель измерения расстояний между датчиками и класс `SensorAnimator` для визуализации динамики системы в реальном времени.

Параметры системы задаются через `SystemParams`:
- `N` — количество верёвок
- `M` — датчиков на верёвку
- `L` — длина сегмента между датчиками (м)
- `d` — расстояние между точками крепления (м)
- `k`, `gamma` — жёсткость верёвки и сопротивление воды
- `wave_amp` — амплитуда волн

### `Kalman.py`
Расширенный фильтр Калмана (EKF) для оценки положения каждого датчика по шумным измерениям расстояний. Вектор состояния — `[x, y, vx, vy]`. Реализованы как фильтр для одного датчика (`SingleSensorKalman`), так и связанный фильтр для всей сети. Модуль строит траектории, доверительные эллипсы и сравнивает оценки с истинными положениями.

### `Error.py`
Анализ точности триангуляции:
- Матрица чувствительности (Якобиан)
- DOP-факторы (HDOP, VDOP, PDOP) — насколько геометрия сети усиливает ошибку измерений
- Монте-Карло оценка распределения погрешностей
- Тепловые карты ошибок по сети и корреляции между датчиками

```
numpy
matplotlib
scipy
```

## Быстрый старт

```bash
git clone https://github.com/MajorN13/sonar
cd sonar-main/program

# Запуск симуляции с анимацией
python Simulation.py

# Запуск фильтрации Калмана
python Kalman.py

# Анализ погрешностей
python Error.py
```

> Для сохранения анимации в видеофайл потребуется `ffmpeg`.


## Результаты

Папка `images/` содержит готовые графики: траектории датчиков, тепловые карты ошибок, корреляции, анализ чувствительности и сравнение работы фильтра Калмана

## Анимации

Папка `Animations/` содержит видеозаписи работы симуляции:

| Файл | Описание |
|------|----------|
| `sonar_basic.mp4` | Физическая симуляция сети датчиков: движение верёвок под воздействием волн и течения |
| `sonar_triangulation.mp4` | Симуляция с триангуляцией и предсказанием положений датчиков в реальном времени |
